Instructions for setup

1. Unzip the file
2. Setup .env file and create a database
3. Run `php artisan migrate` to create the tables
4. Run `php artisan migrate:fresh --seed` to populate the tables using the factory seeder
5. Run `composer install` to install php packages/dependencies
6. Run `npm install` to install javascript packages/dependencies
7. Run `npm run dev` to compile vue assets/components
8. Run `php artisan serve` and navigate to `/` to see the results.
